#yirua
A mini and ease-to-use templating plugin for `jQuery` or `Zepto`.
<br>
<br>

##开始使用(Start)：
####1.引入JQuery或者Zepto文件。
```html
<script src="jquery.min.js"></script>
```
Or
```html
<script src="zepto.min.js"></script>
```


####2.引入jquery.yirua.js。
```html
<script src="jquery.yirua.min.js"></script>
```
