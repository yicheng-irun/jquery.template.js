#yirua
A mini and ease-to-use templating plugin for `jQuery` or `Zepto`.<br>
一个基于jQuery或者Zepto的模板框架的插件。压缩后只有3kb
<br>
<br>

##开始使用(Start)：
####1.引入JQuery或者Zepto文件。
```html
<script src="jquery.min.js"></script>
```
Or
```html
<script src="zepto.min.js"></script>
```


####2.引入jquery.yirua.js。
```html
<script src="jquery.yirua.min.js"></script>
```

####3.你现在获得类似这样的结构
